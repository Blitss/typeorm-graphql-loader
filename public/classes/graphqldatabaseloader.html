Class GraphQLDatabaseLoader

This is the base class for the loader. An instance of this class should be passed to your request context. It is advised that a new instance be created for every request to prevent data leaking.

example
import { GraphQLDatabaseLoader } from '@mando75/typeorm-graphql-loader';
const connection = createConnection({...}); // Create your TypeORM connection

const apolloServer = new ApolloServer({
  schema,
  context: {
    loader: new GraphQLDatabaseLoader(connection, {/** additional options if needed**})
  },
});
